# Portfolio Images

इस folder में portfolio images रखें। निम्नलिखित filenames use करें:

## Image Filenames:

1. **digital-marketing-website.png** - Digital Marketing website के लिए
2. **cloud-website.png** - Cloud website के लिए
3. **web-development.png** - Web Development के लिए
4. **course-selling-website.png** - Course Selling website के लिए
5. **registrar-panel.png** - Registrar Panel के लिए
6. **food-deals.png** - Food Deals के लिए

## Format:
- File format: PNG या JPG
- Recommended size: 800x600 pixels या उससे बड़ा
- File names exactly ऊपर दिए गए नामों जैसे होने चाहिए (lowercase, hyphens)

## Example:
```
frontend/public/portfolio/
  ├── digital-marketing-website.png
  ├── cloud-website.png
  ├── web-development.png
  ├── course-selling-website.png
  ├── registrar-panel.png
  └── food-deals.png
```
